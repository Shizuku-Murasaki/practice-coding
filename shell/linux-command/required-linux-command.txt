●必修 linux command

$ ps : 現在、Linux上で動作しているプロセスを確認する
(オプション)
a : 他の端末を含め、端末操作のプロセスを表示する
-x : 現在実行しているプロセスのみを表示する
c : 実行しているコマンド名を表示する
f : プロセスを階層で表示する


$ curl : 様々なプロトコル（通信手順）を用いてURLで示されるネットワーク上の場所との間でデータの送受信を行う
(オプション)
-G http://example.com : HTTPリクエストをGETメソッドで送信する
-o file.txt http://www.example.com : 標準出力で表示する代わりに、すべてのデータをファイルに書き込む


$ top : CPU利用率やメモリ使用量、稼働中のプロセスが表示される
(オプション)
-p プロセスID : 監視するプロセスIDを指定します


$ grep 検索正規表現 ファイル名 : ファイル中の文字列を検索する
$ grep 検索文字列 ファイル名 | grep 検索文字列2 : ２つ以上の条件を設定し、どちらにも当てはまる行を表示する
(オプション)
-i : 大文字と小文字を区別せず検索する
-v : 一致しないものを検索する
-l : 検索結果にファイル名のみ表示する
-C : 検索結果に一致した箇所から前後に指定した行数表示する
-r : ディレクトリ内も検索対象とする

$ tail ファイル名 : ファイルの最終行から数行を表示する。標準では１０行を表示する。
(オプション)
-c : 出力する文字数を指定する
-n : 出力する行数を指定する


$ find ファイル名 : ファイルやディレクトリを検索する
(オプション)
判別式 -name : ワイルドカードを使用して、ファイルやディレクトリ名の一部のみ指定した検索ができる
判別式 -type f : ファイルのみを対象とし検索する
判別式 -type d : ディレクトリを対象とし検索する


$ chmod アクセス権限 ファイル名 : ファイルやフォルダのパーミッション、アクセス権限を設定する
(オプション)
-v : コマンド実行の診断結果を表示する
-R : 複数ファイルに対して設定する場合、ディレクトリ内も設定対象とする


$ chown 所有者名 ファイル名またはディレクトリ名 : ファイルやディレクトリの所有者を変更する
(オプション)
-R : ディレクトリ内の所有者も変更する
-c : 変更があったとき、メッセージを表示する


$ コマンド & : コマンドをバックグランドジョブとして実行する

$ コマンド1 && コマンド2 : コマンド複数をAND演算で実行する
